# kufar-upgrade-assistant

## Описание проекта по структуре из урока

### Название
**kufar-upgrade-assistant** — агент подбора б/у комплекта «материнская плата + процессор» на kufar.by.

### Проблема
Ручной поиск совместимой платы и CPU на сайте объявлений занимает много времени. Нужно сверять сокет, поколение, чипсет, тип памяти, PCIe, отбрасывать брак, noname и лоты без цены, а потом сравнивать цены — десятки однотипных действий.

### Пользователь
Домашний пользователь ПК, который хочет недорого модернизировать машину, в том числе под локальные LLM-модели и современную видеокарту.

### Основная функция
- Вход: текущая конфигурация ПК и пожелания по апгрейду.
- Сбор: актуальные объявления с kufar.by по заданным категориям и региону.
- Фильтр: DDR4, PCIe ≥ 4.0, whitelist-бренды, без дефектов, с числовой ценой, регион Минск.
- Совместимость: сочетание платы и CPU из разных объявлений с проверкой сокет + поколение + чипсет.
- Выход: топ-3 Intel и топ-3 AMD с ценами, состоянием, ссылками, заметками.

### Формат результата
Markdown-отчёт с краткой сводкой, таблицами топ-3, сравнением Intel vs AMD, исключёнными лотами и открытыми вопросами. Зеркальные JSON/CSV для импорта.

### Минимальный функционал (MVP)
- Intel LGA1700 + DDR4 + PCIe 4.0/5.0.
- AMD AM4 + DDR4 + PCIe 4.0.
- Фильтрация, совместимость, ранжирование топ-3.
- Сохранение отчёта + JSON.
- Проверка актуальности прошлых отчётов.

### Технологии
- Python 3: `requests`, `beautifulsoup4`, `lxml`, `markdown`.
- Claude Code: оркестратор + агенты `kufar-listings-fetcher` и `kit-analyzer`.
- Markdown + JSON + CSV.
- GitHub: репозиторий + Claude skill.
